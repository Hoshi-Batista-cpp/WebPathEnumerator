### WebPathEnumerator 🛡️

Мой проект представляет aсинхронный инструмент на Python для поиска скрытых директорий и файлов на веб-ресурсах. Создан для образовательных целей.

Это мой первый такой проект в такой области. Я постараюсь как можно чаще его улучшать и добавлять новые изменения.

## Особенности
- **Высокая скорость:** Использование `aiohttp` и `asyncio` для параллельных запросов.
- **Маскировка:** Настроенные HTTP-заголовки (User-Agent) для обхода базовых проверок.
- **Гибкость:** Простая структура для работы с любыми словарями.

## Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Hoshi-Batista-cpp/WebPathEnumerator

2. Установить зависимости:
   ```bash
   pip install -r requirements.txt

3. Запустите инструмент, указав адрес сайта:
   ```bash
   python3 fuzzer.py https://github.com

## Wordlists
В папке `WordList/` содержатся списки, основанные на данных из проекта **SecLists**. Вы можете добавлять свои собственные `.txt` файлы в эту папку для расширения области сканирования.
